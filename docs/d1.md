# DELI 1 - Process Creation via Win32 API

## Objective
Learn and implement Windows process creation as the execution backbone of a
custom command-line shell.

## Learnings
- How to use `CreateProcessA` 
- Why to initialize `STARTUPINFO.cb`
- Waiting for Process signal via `WaitForSingleObject`
- Resource cleanup via `CloseHandle`

## Outcome
Successfully launched `notepad.exe` using the CreateProcessA
