# DELI 11 - Win32 UI design and Chloe to Library transformation started

## Objective
make a GUI for chloe to live in

- Creating a Win32 window
- Input and output handling in UI
- Converting chloe from a looping terminal talking program into a callable func that the UI can use

- Win 32 Basics : 
    - Window class registration, Message loop, Lot of Windows messages etc.
    - Edit control is a predefined window class "EDIT"
    - used 2 controls
        - Single Line, tex input, with focus : `hwndInput` 
        - Multi Line, scrollable, readonly : `hwndOutput`
    - `ENTER` listening fiasco in `hwndInput` - Subclassed `hwndInput` and detected thru `WM_CHAR as'\r'` coz focus makes the input text field recieve the enter as '\r' instead of `WM_KEYDOWN and VK_RETURN`
    - Added Emoticons to the output, will expand this further to add diff emoticons based on command status
    - have to use `gcc -municode ui.c -o ui;./ui` to run UI, and need to be on a gcc version that has UNICODE support, i wasn't.
